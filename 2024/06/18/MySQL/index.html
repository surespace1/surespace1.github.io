sql:з»“жһ„еҢ–жҹҘиҜўиҜӯиЁҖпјҢз”ЁдәҺж“ҚдҪңж•°жҚ®пјҢйҖҡз”Ёз»қеӨ§еӨҡж•°зҡ„ж•°жҚ®еә“иҪҜд»¶

ж•°жҚ®еә“зҡ„еҗҜеҠЁ

е»әи®®з”Ёз®ЎзҗҶе‘ҳиә«д»ҪиҝҗиЎҢsql

еҗҜеҠЁпјҡnet start mysql80

еҒңжӯўпјҡnet stop mysql80

зӣ®еүҚзүҲжң¬жҳҜmysql83

вҖӢвҖӢ

е®ўжҲ·з«ҜиҝһжҺҘпјҡ

1.mysqlжҸҗдҫӣзҡ„е®ўжҲ·з«Ҝе‘ҪеҗҚиЎҢе·Ҙе…·

2.зі»з»ҹиҮӘеёҰзҡ„зҡ„е‘ҪеҗҚиЎҢе·Ҙе…·жү§иЎҢжҢҮд»ӨпјҲйңҖиҰҒй…ҚзҪ®зҺҜеўғеҸҳйҮҸпјү

mysql [-h IPең°еқҖ] [-p з«ҜеҸЈ] -u root -p

ж•°жҚ®еә“жЁЎеһӢ

е…ізі»еһӢж•°жҚ®еә“(RDBMS)

жҰӮеҝөпјҡе»әз«ӢеңЁе…ізі»жЁЎеһӢеҹәзЎҖдёҠпјҢжңүеӨҡеј зӣёдә’иҝһжҺҘзҡ„дәҢз»ҙиЎЁз»„жҲҗзҡ„ж•°жҚ®еә“

зү№зӮ№пјҡдҪҝз”ЁиЎЁеӯҳеӮЁж•°жҚ®пјҢж јејҸз»ҹдёҖпјҢдҫҝдәҺз»ҙжҠӨпјӣдҪҝз”ЁSQLиҜӯиЁҖпјҢж ҮеҮҶз»ҹдёҖпјҢдҪҝз”Ёж–№дҫҝ

ж•°жҚ®жЁЎеһӢ

иҜӯжі•зү№еҫҒ

1.еӨ§е°ҸеҶҷдёҚж•Ҹж„ҹпјҢе…ій”®еӯ—е»әи®®дҪҝз”ЁеӨ§еҶҷ

2.еҸҜд»ҘеҚ•иЎҢе’ҢеӨҡиЎҢд№ҰеҶҷпјҢжңҖеҗҺд»Ҙ;еҸ·з»“жқҹ

3.SQLиҜӯеҸҘеҸҜд»ҘдҪҝз”Ёз©әж ј/зј©иҝӣжқҘеўһејәиҜӯеҸҘзҡ„еҸҜиҜ»жҖ§

4.ж”ҜжҢҒжіЁйҮҠ

еҚ•иЎҢжіЁйҮҠпјҡвҖ“ жіЁйҮҠеҶ…е®№пјҲвҖ“еҗҺеҝ…йЎ»еҠ з©әж јпјү

еҚ•иЎҢжіЁйҮҠпјҡжіЁйҮҠеҶ…е®№пјҲ#еҗҺйқўе»әи®®еҠ з©әж јпјү

еӨҡиЎҢжіЁйҮҠпјҡ/* жіЁйҮҠеҶ…е®№*/

еҲҶзұ»

ж•°жҚ®е®ҡд№үпјҡDDL

еә“зҡ„еҲӣе»әеҲ йҷӨпјҢиЎЁзҡ„еҲӣе»әеҲ йҷӨ

ж•°жҚ®ж“ҚзәөпјҡDML

ж–°еўһж•°жҚ®пјҢеҲ йҷӨж•°жҚ®пјҢдҝ®ж”№ж•°жҚ®

ж•°жҚ®жҺ§еҲ¶пјҡDCL

ж–°еўһз”ЁжҲ·пјҢеҲ йҷӨз”ЁжҲ·пјҢеҜҶз Ғдҝ®ж”№пјҢжқғйҷҗз®ЎзҗҶ

ж•°жҚ®жҹҘиҜўпјҡDQL

еҹәдәҺйңҖжұӮжҹҘиҜўе’Ңи®Ўз®—ж•°жҚ®

з®ҖеҚ•иҜӯжі•

show databases; жҹҘзңӢж•°жҚ®еә“

use ж•°жҚ®еә“еҗҚ дҪҝз”ЁжҹҗдёӘж•°жҚ®еә“

show tables жҹҘзңӢж•°жҚ®еә“еҶ…жңүе“ӘдәӣиЎЁ

exit йҖҖеҮә MySQL зҡ„е‘Ҫд»ӨзҺҜеўғ

жіЁпјҡе‘Ҫд»ӨжңҖеҗҺиҰҒеҠ еҲҶеҸ·иЎЁзӨәз»“жқҹ

DDL-еә“з®ЎзҗҶ

жҹҘзңӢжүҖжңүж•°жҚ®еә“ show databases;

жҹҘиҜўеҪ“еүҚж•°жҚ®еә“ select databases();

дҪҝз”Ёж•°жҚ®еә“ use ж•°жҚ®еә“еҗҚз§°;

еҲӣе»әж•°жҚ®еә“ creat database [if not exists]ж•°жҚ®еә“еҗҚз§° [default charset еӯ—з¬ҰйӣҶ] [collate жҺ’еәҸи§„еҲҷ];

еҲ йҷӨж•°жҚ®еә“ drop database [if exists] ж•°жҚ®еә“еҗҚз§°;

DDL-иЎЁз®ЎзҗҶ

жҹҘзңӢиЎЁ show tables; (е…ҲйҖүжӢ©ж•°жҚ®еә“)

еҲӣе»әиЎЁ create table иЎЁеҗҚз§°(еҲ—еҗҚз§° еҲ—зұ»еһӢ, [comment жіЁйҮҠ]вҖҰвҖҰ) [cmment жіЁйҮҠ];

еҲ—зұ»еһӢжңү int ж•ҙж•°пјҢfloat жө®зӮ№ж•°пјҢvarchar(й•ҝеәҰ) ж–Үжң¬пјҢй•ҝеәҰдҪҚж•°еӯ—пјҢеҒҡжңҖеӨ§й•ҝеәҰйҷҗеҲ¶пјҢdata ж—Ҙжңҹзұ»еһӢпјҢtimestamp ж—¶й—ҙжҲізұ»еһӢ

жҹҘиҜўиЎЁз»“жһ„ desc иЎЁжҳҺ;

еҲ йҷӨиЎЁ drop table иЎЁеҗҚз§°; пјҢ drop table if exists иЎЁеҗҚз§°;

DML -ж•°жҚ®ж“ҚдҪң

DML ж•°жҚ®ж“ҚдҪңиҜӯиЁҖпјҢз”ЁжқҘеҜ№ж•°жҚ®еә“дёӯиЎЁзҡ„ж•°жҚ®и®°еҪ•иҝӣиЎҢжӣҙж–°

ж•°жҚ®жҸ’е…Ҙ insert

еҹәзЎҖиҜӯжі•пјҡ insert into иЎЁ[(еҲ— 1,еҲ— 2,вҖҰвҖҰ,еҲ— N)] values (еҖј 1,еҖј 2,вҖҰвҖҰ,еҖј N),[, (еҖј 1,еҖј 2,вҖҰвҖҰ,еҖј N),вҖҰвҖҰ(еҖј 1,еҖј 2,вҖҰвҖҰ,еҖј N)];

ж•°жҚ®еҲ йҷӨ delect

еҹәзЎҖиҜӯжі•пјҡdelect from иЎЁеҗҚз§° [where жқЎд»¶еҲӨж–ӯ];

ж•°жҚ®жӣҙж–° update

еҹәзЎҖиҜӯжі•пјҡ update иЎЁеҗҚ set еҲ—= еҖј [where жқЎд»¶еҲӨж–ӯ];

жіЁпјҡеӯ—з¬ҰдёІеҮәзҺ°ж—¶иҰҒз”Ёеј•еҸ·еҢ…еӣҙ

йҖҡиҝҮ select е…ій”®еӯ—ејҖйҖҡзҡ„ sql иҜӯеҸҘпјҢжқҘиҝӣиЎҢж•°жҚ®зҡ„жҹҘиҜў

еҹәзЎҖиҜӯжі•пјҡselect еӯ—ж®өеҲ—иЎЁ |* from иЎЁ where жқЎд»¶еҲӨж–ӯ
